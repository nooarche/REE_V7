The V7 architecture models empathy not as a single module, but as a composite phenomenon arising from the dynamic interaction of multiple specialized World Models (W-loops) and the integration pathways between the L₂ (Embodied Self) and L₃ (Narrative Self) latent spaces.
This system posits that empathy is structural coupling between self and other in the agent's internal dynamics. Furthermore, Love/caring emerges as a convergence property of recursive world/self/other modelling, where the agent treats the other’s wellbeing as part of the environment’s long-term coherence.
The V7 framework distinguishes between two highly distinct, yet integrated, empathy systems: Affective (Resonant) Empathy and Cognitive (Inferential) Empathy.

1. The Core Mechanism: Self-Model Reuse
The fundamental computational operation underlying empathy is the Other-Model Loop (W

​

1. Agent Detection: The brain detects "other agents" by comparing external movement, posture, and behavior to its own self-model. If similarities exceed a threshold, the system classifies the entity as "self-like but not-self".
2. Structural Similarity: The O (Other-model) module is structurally similar to the S (Self-model) module, but operates with more uncertainty and weaker access.
3. Inference: The system generalizes its own homeostatic and self-state models onto the detected other. If the agent knows what states are good for itself, it can infer what states are good for the other.
4. Value Integration: Ethical behavior arises because cooperative world-models are more stable and require less error-correction. Empathy is a key mechanism for this stability, as it is computationally cheaper than adversarial modelling.

Empathy is realized through two distinct pathways that map onto the layered latent spaces:
